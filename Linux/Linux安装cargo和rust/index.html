Linux安装cargo和rust

首页 Linux

rust安装，因为被墙的原因，在国内几乎很难安装，需要通过代理安装。但是rustup却很容易，一般在linux下，通过官方指定的下列语句，基本可以安装rustup。

curl -sSf https://sh.rustup.rs | sh

安装

然后我们选择1默认安装

待他下载完毕之后，此时我们新建一个窗口，然后添加环境变量:

"$HOME/.cargo/env"

再测试一下发现都下载好了

此时我们配置一个国内rust源，提高依赖下载速度,只需要在.cargo文件夹下新建一个文件config配置一下。

nano /root/.cargo/config

添加进去如下

[source.crates-io]
registry = “https://github.com/rust-lang/crates.io-index"

#指定镜像

replace-with = ‘镜像源名’ # 如：tuna、sjtu、ustc，或者 rustcc

#注：以下源配置一个即可，无需全部

#中国科学技术大学

[source.ustc]
registry = “git://mirrors.ustc.edu.cn/crates.io-index”

#上海交通大学

[source.sjtu]
registry = “https://mirrors.sjtug.sjtu.edu.cn/git/crates.io-index"

#清华大学

[source.tuna]
registry = “https://mirrors.tuna.tsinghua.edu.cn/git/crates.io-index.git"

#rustcc社区

[source.rustcc]
registry = “https://code.aliyun.com/rustcc/crates.io-index.git"

然后Ctrl O保存 Ctrl X退出
注：如果出现保存被拒绝的情况，请切换到root身份！

现在我们可以放心使用cargo 和rustup命令了
