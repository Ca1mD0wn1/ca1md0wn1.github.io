使用临时堆栈对堆栈进行排序

给定一堆整数，使用另一个临时堆栈按升序对其进行排序。

例子：

输入 ：[34、3、31、98、92、23]
输出：[3、23、31、34、92、98]

输入：[3、5、1、4、2、8]
输出：[1、2、3、4、5、8]

算法：

1、创建一个临时堆栈为tmpStack
2、虽然输入堆栈不为空，但请执行以下操作：

  • 从输入堆栈弹出一个元素，成为temp
  • 当临时堆栈不为空且临时堆栈顶部大于 temp 时， 临时堆栈中弹出并将其推送到输入堆栈
  • 将temp推入临时堆栈

3、排序后的数字在tempStack

执行：

function sortStack(input){

    let tempStack = [];
    while(input.length > 0){

        let temp = input.pop();

    while(tempStack.length > 0 && tempStack[tempStack.length-1]>temp){

        input.push(tempStack[tempStack.length-1]);

        tempStack.pop();
    }   

    tempStack.push(temp);

    }
    return tempStack;
}


const arr = [34, 3, 31, 98, 92, 23]

console.log(sortStack(arr))

时间复杂度： O(n 2 )其中 n 是给定堆栈中的整数总数。
辅助空间： O(n)
